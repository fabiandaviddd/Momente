# Momente

Morgen-App für Fabian. Ein Button, ein Tap (Arbeit/frei), ein frisch generierter Text, der Momentum aufbaut.

- PWA: in Safari öffnen → Teilen → "Zum Home-Bildschirm"
- API-Key wird nur lokal auf dem Gerät gespeichert (localStorage), nie im Repo
- Ohne Netz oder ohne Key: eingebauter Text-Vorrat
- Kein Tracking, kein Verlauf
